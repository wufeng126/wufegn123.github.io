# DESIGN.md

## 气质与意象
建筑劳务管理系统，企业业务后台风格。专业、克制、数据驱动。减少装饰性元素，强调信息密度和业务闭环。暖灰底色柔和不刺眼，白色卡片区域层次分明。

## 信息架构（按业务流程组织）
- 项目中心：项目管理、证件管理
- 人力工资：花名册、月度工资、工资发放
- 供应商结算：供应商管理、合同管理、结算记录
- 甲方资金：甲方报量、甲方回款
- 数据看板：成本利润中心、供应商成本应付、工人成本统计、资金综合管理
- 系统管理：通知中心、权限管理、系统配置

## Design Tokens

### 色彩
- 背景: #F0F2F5（暖灰，非冷灰）
- 卡片: #FFFFFF
- 边框: #E8EAED
- 主色蓝: #165DFF
- 主色浅蓝: #4080FF
- 绿色增长: #00B42A
- 红色下降: #F53F3F
- 橙色警告: #FF7D00
- 灰色辅助: #86909C
- 侧边栏激活背景: #F0F5FF
- 侧边栏激活边框: #165DFF
- 图表色板: ['#165DFF', '#00B42A', '#FF7D00', '#F53F3F', '#86909C', '#722ED1', '#0FC6C2', '#F77234', '#F5319D', '#3491FA']

### 颜色语义规范
- 产值/收入 → 蓝色系 (#165DFF)
- 成本 → 橙色系 (#FF7D00)
- 利润 → 绿色系 (#00B42A)
- 预警 → 红色系 (#F53F3F)

### 圆角
- 全局: 0.625rem (10px)
- 按钮: 0.5rem (8px)
- 输入框: 0.5rem (8px)

### 阴影
- 卡片: 0 2px 8px rgba(0,0,0,0.04)
- 卡片悬停: 0 4px 12px rgba(0,0,0,0.06)
- 弹窗: 0 8px 24px rgba(0,0,0,0.12)

### 字体
- 数字等宽: font-variant-numeric: tabular-nums
- 数值列右对齐

## 图表
- 图表库: echarts + echarts-for-react
- 渐变面积图: 折线图底部渐变填充
- 柱状图圆角: borderRadius [4,4,0,0]
- tooltip: 十字准线 + 数据弹窗
- 首次加载入场动画: animationDuration 800ms
- 响应式: 随窗口大小自适应

## 侧边栏
- Logo区深蓝渐变背景 (#0F172A → #1E293B)，白色文字
- 菜单项左侧2px蓝色激活指示条
- 父级菜单可展开/折叠，带旋转箭头动画
- 子菜单缩进16px，左侧灰色竖线连接
- 底部固定"系统配置"和"退出"按钮
- 支持折叠为图标模式（宽度64px）

## 顶部导航栏
- 白色底 + 底部1px极细分割线
- 左侧当前页面标题
- 右侧：项目切换下拉框 + 通知铃铛(带红点) + 用户头像
- 高度56px，固定顶部

## KPI指标卡
- 左上角渐变色图标容器（48x48圆角方形）
- 数值28px加粗 + 千分位分隔
- 环比变化绿色↑/红色↓
- 底部迷你趋势图(sparkline)或进度条
- 悬停上移3px + 阴影加深

## 表格
- 斑马条纹（奇数行白色、偶数行#FAFBFC）
- 固定表头(sticky header)
- 行hover蓝色背景淡入
- 数值列右对齐+等宽字体
- 操作按钮默认半透明，hover行时变为可见
- 空数据显示插画+引导文字
- 加载时显示骨架屏

## 表单
- 输入框聚焦时蓝色边框 + 微弱蓝色发光阴影
- 字段分组：相关字段用浅灰背景(#FAFBFC)区隔
- 金额字段自动千分位格式化
- 提交按钮增加加载状态(spinner)

## 动画
- 页面内容加载: fadeInUp (opacity 0→1, translateY 10px→0, 0.3s)
- 卡片悬停: 上移3px + 阴影加深 + 过渡0.2s
- 弹窗打开: 背景模糊 + 缩放0.95→1
- 数字切换时使用animate-pulse短暂高亮
- 页面内容切换: page-content-enter (0.2s, opacity 0.6→1, translateY 4px→0)
- 侧边栏展开/折叠旋转箭头动画

## 交互规范

### 加载状态
- 骨架屏(Skeleton)替代loading文字
- 主页面: 4格骨架卡片 + 图表区骨架

### 数据异常
- 工资异常高亮: 超上月30%标红
- 空状态: 图标+提示文字
- 错误状态: 重试按钮

## 数据展示规范

### 金额格式
- 看板KPI卡片：万元/亿元自适应（<1万显示元，1万~1亿显示万元，>1亿显示亿元）
- 表格金额：千分位分隔 + 2位小数 + ¥前缀
- 百分比：保留1位小数（如 85.3%）

### KPI卡片
- 必须展示：指标名称、数值、单位、统计范围
- 可选：同比/环比、计算口径tooltip（?图标）、风险标识
- 风险标识：回款率超100%→橙色警告、利润为负→红色、成本超收入→红色

### 图表规范
- 所有图表必须有标题、单位、图例
- 空态：图标+提示文字，加载态：骨架屏，错误态：重试按钮
- 长项目名省略号+tooltip，禁止标签重叠
- 最近更新时间显示在图表右上角

### 资金展示逻辑
- 已回款>结算金额时：不显示"负未回款"，改为"超收/预收金额"
- 字段拆分：应收金额、已收金额、未收金额、超收/预收金额、回款率

## 设计禁忌
- 禁止使用冷灰背景，必须使用暖灰(#F0F2F5)
- 禁止使用科技蓝+圆角卡片+蓝紫色渐变这类万能但无趣的组合
- 图表颜色不可超出定义色板范围
- 禁止使用alert()，统一使用toast()
- 禁止直接import echarts/core，统一使用EChartsWrapper

## 业务组件规范
- 状态标签: StatusTag (type: active/completed/suspended/paid/pending/overdue/voided/default)
- 金额显示: AmountDisplay (自动千分位+¥前缀，color: primary/cost/profit/danger/muted)
- 空状态: EmptyState (icon + title + description)
- 确认弹窗: ConfirmDialog (title + description + confirmText + variant)
- 页面标题: 统一使用 text-xl font-semibold tracking-tight
