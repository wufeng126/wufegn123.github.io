# 建筑劳务企业数据管理系统 - 操作手册

> 版本：v1.0.0 | 更新时间：2026年4月1日

---

## 一、系统概述

### 1.1 系统简介
本系统专为建筑劳务企业设计，实现工人成本管理、工程量统计、甲方报量、付款情况等核心业务的数据化管理，帮助企业提升管理效率、降低运营风险。

### 1.2 核心功能模块
| 模块 | 功能说明 |
|------|----------|
| 项目管理 | 项目信息维护、进度跟踪、数据统计 |
| 工人管理 | 花名册、工资录入、证件管理 |
| 工程量统计 | 预算量、对上报量、对下结算量管理 |
| 产值结算 | 甲方报量、开票、回款、税费计算 |
| 成本利润中心 | 成本分析、利润核算、异常预警 |
| 资金管理 | 收款、付款、工资发放、流水统计 |
| 签证/证件管理 | 签证跟踪、证件到期预警 |
| 零星材料 | 零星材料采购统计 |

## 二、核心操作指南

### 2.1 项目管理

#### 新增项目
1. 点击【项目管理】→【新增项目】
2. 填写项目基本信息：
   - 项目名称（必填）
   - 项目地址
   - 合作方
   - 合同金额
   - 项目年份
   - 项目状态（进行中/已完成/暂停）
3. 点击【保存】

#### 项目进度录入
1. 进入项目详情页
2. 点击【工程量】标签
3. 录入各分项工程的预算量、对上报量、对下结算量
4. 系统自动计算完成进度

### 2.2 工人管理

#### 花名册管理
1. 点击【工人管理】→【花名册】
2. 支持单个新增或批量导入工人信息
3. 可按工种、项目筛选查看

#### 工资录入
1. 点击【工人管理】→【工资录入】
2. 选择工人、项目、月份
3. 录入工时、时薪/包工工资
4. 系统自动计算应发/实发工资
5. 支持批量导入Excel

### 2.3 产值结算

#### 甲方报量录入
1. 点击【产值结算】→【新增结算】
2. 选择项目，录入：
   - 结算金额（甲方确认的工程量金额）
   - 开票金额（本次开票金额）
   - 扣款金额
   - 比例付款
   - 税率（默认继承项目税率）
3. 系统自动计算：
   - 不含税收入 = 开票金额 / (1 + 税率/100)
   - 税费 = 开票金额 - 不含税收入

#### 回款管理
1. 点击【资金管理】→【收款记录】
2. 录入甲方付款信息
3. 系统自动计算待收款金额

### 2.4 成本利润中心

#### 查看成本分析
1. 点击【成本利润中心】
2. 查看各项目成本构成：
   - 人工费（工人工资）
   - 材料机械费（供应商结算）
   - 综合费用
   - 税费
   - 零星材料费

#### 核心计算公式
```
总成本 = 人工费 + 材料机械费 + 综合费用 + 税费 + 零星材料费
含税收入 = 开票金额 + 签证金额
项目利润 = 含税收入 - 总成本
利润率 = 利润 / 含税收入 × 100%
```

### 2.5 零星材料管理

#### 录入零星材料
1. 点击【零星材料】→【新增材料】
2. 录入材料信息：
   - 所属项目
   - 材料名称
   - 数量、单价
   - 采购日期
3. 系统自动计算金额 = 数量 × 单价
4. 金额自动计入项目材料成本

## 三、预警功能说明

### 3.1 证件到期预警
- **即将到期**：距离到期日150天内（橙色提示）
- **已过期**：已超过到期日（红色警告）
- 首页实时显示预警数量，点击可跳转处理

### 3.2 成本超支预警
- 项目利润为负时触发红色预警
- 成本超过收入时触发橙色预警

### 3.3 进度预警
- 对上报量进度超过80%时提醒
- 对上报量超过预算时触发超预算预警

## 四、数据导入导出

### 4.1 Excel导入规范
| 模块 | 支持导入 | 模板下载 |
|------|----------|----------|
| 工人花名册 | ✓ | 花名册页面下载 |
| 工人工资 | ✓ | 工资录入页面下载 |

### 4.2 导入注意事项
1. 严格使用系统提供的模板
2. 日期格式：YYYY-MM-DD
3. 金额格式：数字，不带货币符号
4. 项目名称必须与系统中的项目名称完全一致

## 五、常见问题解答

### Q1: 项目利润计算不准确？
**A**: 检查以下数据是否完整：
- 甲方报量中的开票金额
- 工人工资是否已录入
- 供应商结算是否已录入
- 综合费用是否已录入
- 税率是否正确设置

### Q2: 证件预警不显示？
**A**: 
1. 确认证件已录入到期日期
2. 刷新首页数据
3. 检查系统日期是否正确

### Q3: 导入数据失败？
**A**: 
1. 检查文件格式是否为CSV或XLSX
2. 检查必填字段是否完整
3. 检查数据格式是否符合要求
4. 查看错误提示，按提示修正

### Q4: 数据统计与实际不符？
**A**: 
1. 确认所有相关数据已录入
2. 检查是否有重复数据
3. 使用项目筛选器确认数据范围
4. 联系管理员核查数据库
