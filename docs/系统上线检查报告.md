## 三、数据逻辑与计算准确性校验

### 3.1 核心计算公式验证 ✅

| 公式 | 验证结果 | 说明 |
|------|----------|------|
| 项目利润 = 含税开票金额 - (人工费 + 机械材料费 + 综合费用 + 税费 + 零星材料费) | ✅ 正确 | 已验证成本利润中心计算逻辑 |
| 待收款金额 = 项目总产值 - 甲方已付款 | ✅ 正确 | 已验证首页待收款计算 |
| 零星材料金额 = 数量 × 单价 | ✅ 正确 | 已验证零星材料表计算 |
| 不含税收入 = 开票金额 / (1 + 税率/100) | ✅ 正确 | 已验证税费计算逻辑 |
| 税费 = 开票金额 - 不含税收入 | ✅ 正确 | 已验证税费自动计算 |
